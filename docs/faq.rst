Frequently Asked Questions (FAQ)
=======================================

This section collects frequently asked questions and answers about Ploggify. If you have any additional questions, please feel free to contact us.

General Questions
----------------------------------

Q1 : What is Ploggify?
    A : **Ploggify** is an app service designed to combine exercise and environmental protection by gamifying "plogging," the activity of picking up litter while jogging.

Q2 : Is this project open source?
    A : Yes, Ploggify is an open source project and can be used, modified, and distributed under the **Apache 2.0** license.
    For more details, please refer to the LICENSE file.

Q3 : What are the main features of Ploggify?
    A : Key features of Ploggify include route recommendation, AI object detection, personal dashboard, and SNS community features.

App Features
----------------------------

Q1 : How does running route recommendation work?
    A : It recommends the optimal plogging route based on the **exercise purpose, goal, and region** information set by the user.

Q2 : What technology is used for AI object detection?
    A : It uses the **YOLO** object detection model provided by Roboflow to categorize and count the types of trash from photos uploaded by the user.

Q3 : What information can I check on the personal dashboard?
    A : In the personal dashboard, you can check statistics such as total running distance and the amount of trash collected.

Q4 : How can I use the SNS community feature?
    A : Users can share their plogging achievements with others and compete through the ranking system.
