Release Notes
=============================

This document records the changes for each version of the Ploggify project.

v0.0.1
-----------------
* **Route Recommendation System**
  
  - Recommends optimal running/plogging routes based on the user's purpose and location.

* **AI Object Detection**
  
  - Uses the **YOLO** model to automatically identify the types of trash photographed (plastic, cans, etc.).
  - Automatically counts and records the number of collected trash items.

* **Personal Dashboard**
  
  - Allows you to check activity statistics, such as total running distance and collected trash amount, at a glance.

* **SNS Community Features**
  
  - Provides basic community features to share plogging achievements and communicate with other users.

.. note::
  This version is an initial development release, so some features may be unstable or performance may be low.
  If you find any bugs, please report them via Github Issues.
